## Task. Configure a SR policy with delay metric

![](images/13.png)

:keyboard: xrd-1:
```bash
config
segment-routing
 traffic-eng
  policy to-xrd2-latency
   color 400 end-point ipv4 2.2.2.2
   candidate-paths
    preference 100
     dynamic
      pcep
      !
      metric
       type delay
      !
     !
    !
   !
  !
 !
!
commit
end
```
