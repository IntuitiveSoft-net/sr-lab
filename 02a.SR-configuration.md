# Annexes: Segment Routing

## SR-MPLS configurations

xrd-1 configuration:
```bash
configure terminal
router isis core
 address-family ipv4 unicast
  segment-routing mpls
 interface Loopback0
  address-family ipv4 unicast
   prefix-sid absolute 16001
commit
```
