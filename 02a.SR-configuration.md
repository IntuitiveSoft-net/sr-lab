# SR-MPLS Configurations

xrd-2 configuration:
```bash
configure terminal
router ospf 1
 mpls traffic-eng router-id 2.2.2.2
 segment-routing mpls
 area 0
  mpls traffic-eng
  segment-routing mpls
  interface Loopback0 passive
  prefix-sid absolute 16002
```
