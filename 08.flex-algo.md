# 08. SR IGP Flexible Algorithm

## SR IGP Flexible Algorithm (FA)

## Task 1. Activate segment-routing on xrd-1

:keyboard: xrd-1

```bash
configure terminal
router isis core
 flex-algo 128
  metric-type te
 interface Loopback0
  address-family ipv4 unicast
   prefix-sid algorithm 128 absolute 16001
commit
end
exit
```
