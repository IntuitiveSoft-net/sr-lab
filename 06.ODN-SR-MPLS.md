Create a VRF on xrd-1 and xrd-2:

:keyboard: xrd-1
```bash
vrf customer1
 address-family ipv4 unicast
  import route-target
   10:10
  !
  export route-target
   10:10
  !
 !
!
interface GigabitEthernet0/0/0/2
 vrf customer1
 no ipv4 address
!
router bgp 65000
 vrf customer1
  rd 65000:1
  address-family ipv4 unicast
   redistribute connected
  !
 !
!
commit
```
